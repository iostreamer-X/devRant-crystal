# devRant-crystal

A crystal wrapper for devRant API

![](https://travis-ci.org/iostreamer-X/devRant-crystal.svg?branch=master)

## Installation


Add this to your application's `shard.yml`:

```yaml
dependencies:
  devRant-crystal:
    github: iostreamer-X/devRant-crystal
```
